# Python-OpenCV-Deep-Learning

# Python for Computer Vision with OpenCV and Deep Learning

This repository is a collection of the most important and advanced projects completed during the course **Python for Computer Vision with OpenCV and Deep Learning**, created by **Jose Portilla**.

## About This Repository

The purpose of this repository is to organize, document, and preserve the main practical projects developed throughout the course. These projects demonstrate how Python can be used together with OpenCV, NumPy, TensorFlow, Keras, and modern deep-learning techniques to analyze images and video data.

The repository focuses on practical implementations rather than serving as a complete copy of the course material. Each project represents an important concept, technique, or computer-vision workflow covered during the training.

## About the Course

The course provides a comprehensive introduction to computer vision using Python and the OpenCV library.

Computer vision makes it possible to analyze, understand, and extract useful information from images and videos. These technologies are widely used in areas such as:

* Self-driving vehicles
* Robotics
* Security and surveillance
* Medical diagnostics
* Social media applications
* Manufacturing and quality control
* Object recognition and tracking
* Artificial intelligence systems

The course begins with numerical processing using NumPy and progresses through image manipulation, video processing, object detection, tracking, deep learning, and state-of-the-art neural networks such as YOLO.


## Learning Objectives

Through the projects included in this repository, the following skills are developed:

* Understand the fundamentals of NumPy.
* Open and manipulate images using NumPy.
* Work with image files using Python and OpenCV.
* Draw shapes, lines, rectangles, circles, and text on images and video frames.
* Apply smoothing, blurring, thresholding, and morphological operations.
* Create and analyze color histograms.
* Open, process, and stream video using OpenCV.
* Detect corners, edges, grids, contours, faces, and other visual features.
* Segment images using the Watershed algorithm.
* Track objects across video frames.
* Understand and apply optical-flow techniques.
* Build image classifiers using Python and deep learning.
* Train neural networks using TensorFlow and Keras.
* Work with custom image datasets.
* Apply modern object-detection models such as YOLO.

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd <repository-folder>
```

Create a Python virtual environment:

```bash
python -m venv .venv
```

Activate the environment on Windows:

```bash
.venv\Scripts\activate
```

Activate the environment on macOS or Linux:

```bash
source .venv/bin/activate
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Some older course projects may depend on previous versions of TensorFlow, Keras, or OpenCV. Review the documentation inside each project directory before running the code.

## Running the Projects

Python scripts can generally be executed with:

```bash
python project_name.py
```

Jupyter notebooks can be opened with:

```bash
jupyter notebook
```

For webcam-based projects, verify that the camera is connected and that no other application is currently using it.
